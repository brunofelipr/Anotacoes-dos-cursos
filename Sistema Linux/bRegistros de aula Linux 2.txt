ifconfig- encontrado em pacotes no ubuntu- serve para conferir interface de rede
hostname- informações o host
hostname -I- IP
hostname -i- loopback
who- mostra como estamos logados na rede
whoami- nome de login do usuário
ping- verifica se host esta ativo
ping --- (numero)- limita o numero de pacotes a serem recebidos
traceroute- traça a rota de rede
whois- traz informações de servidor dns, domínio na internet e muito mais sobre um endereço
finger- informações sobre usuário logado no host
dig host- informações sobre o dns de um host
dig host +short- numero de ip de um dns ou host
w- informações de who mais detalhadas
history -c- limpa histórico
alias (sequencia de letras à sua escolha) + '(comando de sua escolha)'- da um atalho de letras à sua escolha para um comando
nl- conta quantidade de linhas de uma arquivo.txt
wc -l- linhas escritas e em branco
wc -w- conta palavras de um texto
wc -c- bytes de um arquivo
wc -m- caractéres de um arquivo
ls -a- arquivos ocultos
sort -n - organiza em ordem numérica arquivo txt
last reboot- mostra todas as informações de reinicialização do sistema
route -n- tabelas de roteamento IP do kernel: interface, roteador, mascara, etc...
time- mostra o tempo de um processo
uptime- mostra o tempo que o sistema está rodando
cowsay- mostra uma vaca falando a mensagem que você escrever
cmatrix- efeito matrix
init 0- desliga a máquina
halt- desliga a máquina
cmp- compara dois arquivos
seq- imprime contagem de números com inicio e fim
whereis- exibe o caminho do programa e seu manual
which- exibe o caminho de um programa
logout- finaliza sessão

Pacotes-
programas colocados dentro de um arquivo identificados por sua extensão, incluindo arquivos necessários para instalações de programas
Gerenciadores de pacotes-
possuem resolução automática de dependência entre pacotes
